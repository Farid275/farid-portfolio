# Farid Portfolio

A professional developer portfolio built with Next.js, TypeScript, Tailwind CSS, and Motion.

## Features

- Clean responsive layout
- Animated hero section
- Scroll reveal animation
- Interactive project cards
- Featured project case study
- Tech stack section
- Contact section

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Motion
- Vercel

## Main Project

EduManage is presented as the main featured project, focused on course management, dashboard structure, CRUD features, and database integration.
