# Farid Portfolio

A personal software engineering and full-stack developer portfolio built with Next.js, TypeScript, Tailwind CSS, and Motion.

## Features

- Responsive portfolio layout
- Smooth scroll animation
- Animated section reveal
- Project showcase
- Education section
- Blog preview section
- Contact and social links

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Motion
- Lenis

## Getting Started

```bash
npm install
npm run dev
