# DESIGN.md

## Portfolio Direction
This portfolio is for a software engineering / full-stack developer profile.

The design should feel:
- Clean
- Modern
- Professional
- Structured
- Credible
- Slightly animated, but not excessive

Avoid:
- Generic AI landing page style
- Excessive gradients
- Neon colors
- Overly playful effects
- Too many floating elements
- Fake experience or exaggerated claims

## Visual Identity
Primary color:
- Deep navy / dark blue

Accent color:
- Soft yellow / warm gold

Background:
- White or very light neutral

Text:
- Near black / dark slate

Border:
- Soft gray

## Layout Style
Use:
- Spacious layout
- Clear section separation
- Card-based project layout
- Smooth scroll
- Responsive mobile layout
- Consistent spacing

## Animation Direction
The website should not feel static.

Use subtle animations:
- Hero text fade-up
- Section reveal on scroll
- Project card hover lift
- Button hover transition
- Small background floating gradient
- Tech badges slight hover effect

Avoid:
- Too much movement
- Fast animation
- Distracting animation
- Continuous heavy animation
- Animation that makes text hard to read

## Main Sections
The portfolio must include:
1. Navbar
2. Hero section
3. About section
4. Tech Stack section
5. Featured Projects section
6. EduManage Case Study section
7. Education section
8. Contact section
9. Footer

## Important Rules
- Do not introduce random colors.
- Do not redesign the whole website without instruction.
- Do not invent fake experience.
- Keep the portfolio suitable for professional and academic review.
